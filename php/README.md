# dhdjencode For PHP

dhdjencode是一个高性能的高安全性加密算法（由著名架构师、算法师dhdj开发）

这个慈善项目(大喊大叫基金会)将会把dhdjencode使用各种语言实现。这必将是算法界的改革。

## How to use

```
<?php

include "dhdjencode.php";

echo dhdjencode("Hacking to the Gate.");

?>

```
